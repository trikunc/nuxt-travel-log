# Nuxt Travel Log

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm install
```

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

## Production

Build the application for production:

```bash
# pnpm
pnpm build
```

Locally preview production build:

```bash
# pnpm
pnpm preview
```
